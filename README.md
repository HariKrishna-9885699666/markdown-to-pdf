# Markdown to PDF

A clean, modern React application for converting Markdown files or text into formatted PDF documents.

**Live Demo:** [https://markdown-to-pdf-react.vercel.app/](https://markdown-to-pdf-react.vercel.app/)

## Tech Stack
- React 19
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zustand (State Management)
- react-markdown
- jspdf / html2canvas (PDF Generation)

## Getting Started

### Prerequisites
- Node.js
- Yarn 4.2.9 or higher

### Installation

1. Install dependencies:
```bash
yarn install
```

2. Start the development server:
```bash
yarn dev
```

3. Build for production:
```bash
yarn build
```

4. Preview production build:
```bash
yarn preview
```

## Features
- Full Markdown support (GFM, tables, code blocks)
- Live side-by-side preview
- One-click PDF export
- Syntax highlighting for code blocks
- Dark / Light mode themes
